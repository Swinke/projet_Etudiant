

Programme réalisé dans le cadre de ma licence 3 MIAGE

--- Détails du programme ---

Implémentation de commandes Unix tels que ls, cd ect...

--- Lancement du programme ---
Sous linux,

Compilation : avec le terminal tapez "gcc TP1_Damand.c -o programme"

Execution : avec le terminal tapez "./programme"



Fonctionnement programme :
 - Vous n'avez plus qu'a suivre les instructions


Pour les fonctions de copie ou déplacement de fichier vous devez stipuler dans
le chemin de destination le nom du fichier (le même ou un autre) sinon le
programme vous indiqueras une erreur.
