---- Détails du projet ----

Création d'IA pour un jeu basé sur Pokemon. Chaque IA s'affronte sur un terrain qui diminue au fur et à mesure,
lorsque plusieurs IA se retrouve sur une même case un tournoi est lancé.

--- Lancement du programme ---

Importez le projet sous votre IDE, puis pour démarrer le programme exécuté le fichier "TestPokemonJava" dans le packtage "game".
Pour executé le programme avec différentes IA modifier "new PowerJoueur(0)" dans la ligne "List<Joueur> jTypes = new ArrayList<>(Arrays.asList(new PowerJoueur(0)));"
par "new NOM_IA(0)", vous trouverez les diiférentes IA dans le packtage "joueurs"
